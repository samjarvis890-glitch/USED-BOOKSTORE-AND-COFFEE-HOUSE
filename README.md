# Pages & Perks: Used Bookstore & Coffee House

A complete, premium, responsive website built for a cozy literary lounge.

---

## 📂 Project Structure

```text
used-bookstore-coffeehouse/
│
├── index.html          # Home Dashboard (Hero, Bestsellers, Cafe Intro, Community, Events, CTA)
├── about.html          # About Our Story (Classic Hero, History Timeline, Pillars, Team Curators, CTA)
├── books.html          # Browse Books (Masonry Genres Grid, Bestsellers, Rare Spotlight, Live Search, CTA)
├── cafe.html           # Cafe Menu (Signature Coffee, Tabbed Interactive Menu, Seating Cards, Gallery, CTA)
├── events.html         # Gathering Calendar (Weekly Table, Guest Authors, Workshops, Open Mic, RSVP CTA)
├── loyalty.html        # Loyalty Club (Digital Interactive Punch Card, Points Tiers, Rankings, Testimonials, CTA)
├── tradein.html        # Trade-In Program (Step Process, Interactive Store Credit Calculator, CTA)
├── contact.html        # Contact & Hours (Quick Cards, Glassmorphic Contact Form, FAQ Accordion, Maps)
├── login.html          # Account Sign-in (Single-screen Glassmorphic portal, No footer)
├── register.html       # Account Registration (Single-screen Glassmorphic form, No footer)
├── 404.html            # Error Page (Centered Spilled Chapter Illustration, No header/footer)
│
├── assets/
│   ├── css/
│   │   ├── style.css         # Custom theme variables, layout rules, animations, LTR/RTL overrides
│   │   ├── dark-mode.css     # Dark Mode variables & input overrides
│   │   ├── responsive.css    # Media query specifications (Mobile to Ultra-Wide)
│   │
│   ├── js/
│   │   ├── main.js           # Live book search, tabbed menu filter, calculator compute, punch card stamps
│   │   ├── darkmode.js       # Persistent dark theme storage & DOM management
│   │   ├── menu.js           # Sticky navbar trigger, mobile menu auto-collapsing, LTR/RTL toggle
│   │   └── animations.js     # Scroll reveal (IntersectionObserver) & 3D tilt effects
│   │
│   └── images/               # Category folders with local images (Zero internet dependency)
```

## 🎨 Brand Design Tokens

---

## ⚡ Interactive Features

1. **Persistent Dark Mode**: Toggle changes colors instantly and stores selections in `localStorage`.
2. **Persistent LTR/RTL Toggle**: Switches text layouts, button positions, and alignments instantly. Stores layout direction state.
3. **Live Search**: Filters books on `books.html` instantly as you type.
4. **Digital Punch Card**: Mock-stamps a customer card; triggers congratulations when full.
5. **Trade-In Calculator**: Computes approximate store credit based on book condition, categories, and volumes.
